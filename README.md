# aitranslator-webapp
A Flask-based web application that translates English text into Hindi using the Helsinki-NLP/opus-mt-en-hi NLP model.

## Setup Hugging Face Token

1. Copy `.env.example` to `.env` in the project root.
2. Put your Hugging Face token in `HF_API_TOKEN`.

Example:

HF_API_TOKEN=hf_xxxxxxxxxxxxxxxxxxxx
